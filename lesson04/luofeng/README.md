### 增加用户
```
-------------------------------
欢迎登录用户系统，请选择对应的操作选项
-------------------------------
    1. 增加用户信息(add)
    2. 删除用户信息(delete)
    3. 更新用户信息(update)
    4. 查询用户信息(query)
    5. 保存用户信息(save)
    6. 退出用户管理系统(exit)
-------------------------------

请输入操作: add
请依次输入用户信息，用户名，年龄，电话号码，地址，以空格分隔: google 28 18000000000 beijing
用户google被成功添加
```
